Lawnchair demo!

We're just going to be mostly playing the with examples located at http://blog.westcoastlogic.com/lawnchair/ in the console

You can switch up the adaptors in demo.js if the loaded adaptor doesn't work in your browser.

There's a reference to the Lawnchair object so you can just type in storage in your console to get a reference.

Api calls

Try giving these call a try with the storage object

  • save storage.save({key:'myKey', value: 'myValue'}, function(r){console.log(r)});
  • get storage.get('myKey', function(r){console.log(r);});
  • find storage.find('r.value == "myValue"', 'console.log(r)');
  • each storage.each('console.log(r)');
  • all storage.all('console.log(r)');
  • nuke storage.nuke();
  • remove storage.remove('myKey');
